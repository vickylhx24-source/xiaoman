# 小满 · 家庭物品与饮食管理

> 一个**纯前端、可离线、零后端依赖**的 PWA，帮你管理家庭库存、菜谱与每日营养摄入。
> 数据全部存在你自己的设备里，不上传任何服务器，隐私完全自主。

🌐 在线地址：**https://vickylhx24-source.github.io/xiaoman/**

---

## ✨ 功能特性

### 📦 库存管理
- 记录家庭物品 / 食材，支持分类整理
- 通过拍照 📷 或扫码 🔍 录入，快捷方便
- 全部数据本地保存，随时可查

### 🍳 菜谱管理
- 收藏菜谱，可**按食材自动分类**（鸡肉类 / 牛肉类 / 猪肉类 / 蛋奶 / 蔬菜 / 其他肉荤…），也支持编辑时**手动调整分类**
- 菜谱图片**大图浏览**（灯箱模式：左右滑动、‹ › 按钮、键盘切换）
- 保存菜谱来源链接，一键在浏览器打开
- 「我的收藏」可按食材分类横向切换，并显示各类数量

### 🥗 营养管理
- 记录每日饮食，支持**按月 / 按日**分组查看
- 点击某一天可查看**当日营养进度**（依据膳食营养素参考摄入量 DRIs 标准展示达标情况）
- 智能识别食材营养素，且**水 / 汤也识别为食材**，不会误报「缺水」

### 💾 数据安全
- 支持**全量备份与恢复**（JSON 导出 / 导入，包含菜谱图片）
- 数据全程保存在本机 IndexedDB，断网也能用

---

## 🛠 技术亮点

- **纯前端 PWA**：HTML / CSS / 原生 JavaScript，无框架、无后端、无数据库服务
- 可「**添加到主屏幕**」，体验接近原生 App
- **离线可用**：Service Worker 缓存资源
- **零 token 依赖**：部署之后免维护、不依赖任何密钥或第三方接口

---

## 📱 使用方式

1. 用手机 / 电脑浏览器打开：**https://vickylhx24-source.github.io/xiaoman/**
2. 浏览器菜单 → **添加到主屏幕**
3. 之后从主屏图标进入即可，和 App 一样用

> **多人 / 多设备**：把同一个网址发给家人即可各自使用，数据在各自设备本地、互不影响；
> **换手机**：在新手机打开网址、导入之前备份的 JSON 即可恢复全部数据。

---

## 🚀 本地运行与部署

- **本地预览**：用任意静态服务器打开 `index.html` 所在目录即可（例如 `python3 -m http.server`）
- **部署上线**：将代码推送到 GitHub，仓库 **Settings → Pages** 选择
  `Deploy from a branch` → `main` 分支、`/ (root)`，保存后即自动发布
